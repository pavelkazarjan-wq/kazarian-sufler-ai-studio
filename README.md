# KazarianWEBINARStudio

**Профессиональная студия для записи вебинаров, онлайн-консультаций и презентаций**

Веб-приложение для психологов, коучей, тренеров и всех, кто проводит онлайн-встречи и хочет записывать их в высоком качестве с автоматической транскрипцией и генерацией протоколов.

**Сайт:** https://kazarian-webinar-ai-studio.netlify.app

---

## Быстрый старт

1. Откройте сайт в браузере Chrome или Edge
2. Загрузите презентацию (PDF или изображения) ИЛИ выберите захват экрана
3. Включите камеру и микрофон (кнопки внизу)
4. Нажмите красную кнопку **REC** для начала записи
5. После записи файл автоматически сохранится на компьютер

---

## Основные возможности

### Запись видео

- **Формат:** MP4 (H.264 видео + AAC аудио)
- **Качество:** 1920x1080 @ 30 кадров/сек
- **Длительность:** без ограничений (запись идёт напрямую на диск)
- **Автосохранение:** настройте папку один раз, записи сохраняются без диалогов

### Источники контента

| Кнопка | Что делает |
|--------|------------|
| **Файл** | Загрузить PDF-презентацию или картинки |
| **Экран** | Захватить любое окно или весь экран |
| **Гость** | Подключить собеседника по видеосвязи |

### Режимы камеры

- **Кругляшок** — ваше видео в углу поверх презентации. Можно менять размер и позицию (4 угла).
- **50/50** — экран делится пополам: вы и контент. Можно выбрать кто слева, кто справа.

## Панель настроек

Находится справа. На мобильном — через кнопку меню (три полоски).

### Оверлеи (видны на записи)

| Название | Описание |
|----------|----------|
| **Часы** | Текущее время в углу |
| **YouTube LIVE** | Красная плашка "LIVE" с числом зрителей |
| **Бегущая строка** | Текст, который прокручивается внизу экрана |
| **Плашка имени** | Ваше имя и должность |

### Заставки

| Заставка | Когда показывается |
|----------|-------------------|
| **Splash** | В самом начале записи (загрузите картинку) |
| **Intro** | Видеоролик после заставки (например, заставка канала) |
| **Outro** | Картинка с текстом в конце записи |

### Транскрипция и протокол

Автоматически расшифровывает запись и создаёт структурированный протокол сессии.

**Как настроить:**
1. Получите API ключ на https://platform.openai.com/api-keys
2. В настройках найдите "OpenAI API ключ"
3. Вставьте ключ (начинается с `sk-`)
4. Нажмите "Сохранить"

**Что получите:**
- Полный текст записи (транскрипция)
- Протокол сессии с темами, выводами, рекомендациями
- Можно скачать как текстовый файл

### Live Audience

Имитация живой аудитории для создания эффекта реального эфира:
- **Комментарии** — появляются автоматически
- **Счётчик зрителей** — плавно растёт

## Трансляция в Zoom

1. Нажмите кнопку **"Зум"** в правом верхнем углу
2. Откроется чистое окно без интерфейса
3. В Zoom выберите "Демонстрация экрана" → "Окно" → выберите это окно
4. Участники Zoom увидят вашу презентацию с камерой и всеми оверлеями

## Онлайн-консультации

### Как пригласить клиента:

1. Нажмите кнопку **"Консультация"**
2. Скопируйте ссылку
3. Отправьте клиенту (WhatsApp, Telegram, email)
4. Когда клиент перейдёт по ссылке и разрешит камеру/микрофон — вы увидите друг друга

## Автосохранение записей

При первой записи браузер спросит, куда сохранить файл. Выберите или создайте папку (например, "Записи вебинаров"). Все следующие записи будут сохраняться туда автоматически.

Сохраняются:
- **Видео:** webinar-ДАТА-ВРЕМЯ.mp4
- **Аудио:** audio-ДАТА-ВРЕМЯ.webm (для транскрипции)

## Частые вопросы

**Сколько можно записывать?**
Без ограничений. Запись идёт прямо на диск, не в память браузера.

**Почему не работает транскрипция?**
Проверьте, что ввели и сохранили OpenAI API ключ в настройках.

**Работает ли без интернета?**
Запись — да. Транскрипция и протоколы — нет, нужен интернет.

**Можно ли записывать на телефоне?**
Технически да, но рекомендуется компьютер для стабильной работы.

**Куда сохраняются записи?**
В папку, которую вы выбрали при первой записи. Обычно в "Загрузки" или созданную вами папку.
