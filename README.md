# KazarianSuflerAIStudio

Профессиональный телесуфлёр с AI генерацией скриптов.

## Функции

- AI-сценарист на базе OpenAI (VSL, видеоскрипты, вебинары)
- Телепромптер с автопрокруткой и голосовым управлением
- Видеообработка: фоны, эффекты, настройка освещения
- Оверлеи, субтитры, бегущая строка
- Звуковые эффекты (аплодисменты, барабанная дробь, смех и др.)
- Zustand для state management

## Деплой

Сайт задеплоен на Netlify: https://kazarian-sufler-ai-studio.netlify.app

## Технологии

- React + TypeScript
- Vite
- Zustand
- Tailwind CSS
