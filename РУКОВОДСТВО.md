## 2. Регистрация и вход

### Регистрация нового пользователя

1. Откройте https://expertpage.pro/login.html
2. Нажмите вкладку **"Регистрация"**
3. Заполните поля:
   - **Полное имя** — как будет отображаться в плашке
   - **Специализация** — психолог, коуч, тренер и т.д.
   - **Email** — для входа и уведомлений

1. Введите email и пароль
2. Поставьте галочку **"Запомнить меня"** чтобы не вводить email повторно
3. Нажмите **"Войти"**

### Восстановление пароля

1. Нажмите **"Забыли пароль?"**
2. Введите email
3. Получите ссылку для сброса пароля

## 5. Онлайн-консультации (1 на 1)

### Создание комнаты (для ведущего)

1. Нажмите кнопку **"Гости"** в нижней панели
2. Выберите вкладку **"Создать комнату"**
3. Нажмите **"Создать комнату"**
4. Система покажет:
   - **Код комнаты** — 6-значный код (например: `ABC123`)
   - **Ссылка для клиента** — полная URL для отправки
5. Отправьте ссылку клиенту через мессенджер

### Формат ссылки для клиента

```
https://kazarian-webinar-ai-studio.netlify.app/app.html?mode=client&session=XXXXXX&name=Имя
```

Параметры:
- `mode=client` — режим клиента
- `session=XXXXXX` — код сессии
- `name=Имя` — имя клиента (опционально)

### Подключение клиента

1. Клиент открывает ссылку
2. Разрешает доступ к камере и микрофону
3. Автоматически подключается к комнате
4. Ведущий видит уведомление "Гость подключается..."
5. После подключения — видео гостя появляется

### Запланированные консультации (из Supabase)

1. Откройте выпадающий список **"Консультации"**
2. Выберите сессию из списка
3. Нажмите **"Начать"**
4. Клиент получает уведомление в Telegram боте

## 6. Групповые сессии (3 участника)

### Подключение второго гостя

1. Первый гость уже подключен (см. раздел 5)
2. Отправьте вторую ссылку с другим именем:
   ```
   https://kazarian-webinar-ai-studio.netlify.app/app.html?mode=client&session=XXXXXX&name=Гость2
   ```
3. Когда второй гость подключится:
   - Автоматически включится **режим 3 участников**
   - Экран разделится на 3 части

### Раскладка 3 участников

**Горизонтальный формат (16:9):**
```
┌─────────┬─────────┬─────────┐
│  Хост   │ Гость 1 │ Гость 2 │
│  (вы)   │         │         │
└─────────┴─────────┴─────────┘
     33%       33%       33%
```

**Вертикальный формат (9:16):**
```
┌─────────────┐
│   Гость 2   │  33%
├─────────────┤
│   Гость 1   │  33%
├─────────────┤
│    Хост     │  33%
└─────────────┘
```

### Управление гостями

- **Максимум гостей:** 2 (итого 3 участника)
- При отключении гостя 2 — возврат к режиму 2 участников
- У каждого гостя своя плашка с именем

## 8. Запись видео

### Типы записи

| Тип | Где сохраняется | Качество |
|-----|-----------------|----------|
| **Локальная** | Браузер → скачивание | WebM/MP4 |
| **Серверная** | Hetzner → скачивание | MP4 (FFmpeg) |

### Локальная запись (в браузере)

1. Нажмите **🔴 Запись**
2. Таймер начнёт отсчёт
3. Для остановки нажмите повторно
4. Файл конвертируется в MP4
5. Автоматически скачивается

### Серверная запись

Используется для длительных записей и лучшего качества.

1. Запись автоматически отправляется на сервер через WebSocket
2. Сервер обрабатывает через FFmpeg
3. По завершении — файл автоматически скачивается
4. Файлы хранятся на сервере 24 часа

**Путь на сервере:** `/opt/stream-server/recordings/`

### Форматы записи

| Параметр | 16:9 | 9:16 |
|----------|------|------|
| **Разрешение** | 1920×1080 | 1080×1920 |
| **FPS** | 24 или 30 | 24 или 30 |
| **Кодек** | H.264 (MP4) | H.264 (MP4) |
| **Аудио** | AAC 128kbps | AAC 128kbps |

## 10. Режиссёрский пульт

### Назначение

Отдельная страница для помощника/продюсера, который управляет оверлеями во время эфира, не имея доступа к камере ведущего.

### Как получить код сессии (ведущий)

1. Нажмите оранжевую кнопку **"Режиссёр"** в шапке
2. Появится всплывающее окно с кодом (например: `P7X2KA`)
3. Скопируйте код или ссылку
4. Отправьте помощнику

### Как подключиться (помощник)

**Вариант 1 — по ссылке:**
```
https://kazarian-webinar-ai-studio.netlify.app/producer.html?session=P7X2KA
```

**Вариант 2 — вручную:**
1. Откройте https://kazarian-webinar-ai-studio.netlify.app/producer.html
2. Введите код сессии
3. Нажмите OK

### Функции пульта

| Кнопка | Действие |
|--------|----------|
| **Плашка ведущего** | Показать/скрыть |
| **Плашка гостя** | Показать/скрыть |
| **Бегущая строка** | Вкл/выкл |
| **CTA кнопка** | Показать/скрыть |
| **🔇 ЦЕНЗУРА** | Зажать для заглушки видео и звука |

### Кнопка цензуры

- Зажмите кнопку — видео размывается, звук отключается
- Отпустите — всё возвращается
- Используйте при технических проблемах или неуместном контенте

### Изоляция сессий

- Каждый ведущий имеет **уникальный код**
- Помощник одного ведущего **не может** управлять студией другого
- Коды меняются при перезагрузке страницы

## 13. Устранение неполадок

### Камера не работает

1. Проверьте разрешения браузера (🔒 в адресной строке)
2. Убедитесь, что камера не занята другим приложением
3. Попробуйте другой браузер (Chrome/Edge рекомендуется)

### Гость не подключается

1. Проверьте правильность ссылки/кода
2. Убедитесь, что оба в одной комнате
3. Проверьте интернет-соединение
4. Попробуйте обновить страницу

### Нет звука при записи

1. Проверьте, включен ли микрофон (кнопка не красная)
2. Проверьте уровень громкости системы
3. Убедитесь, что выбран правильный микрофон в браузере

### Запись не скачивается

1. Дождитесь завершения конвертации
2. Проверьте, не заблокированы ли загрузки
3. Попробуйте серверную запись вместо локальной

### Стрим не запускается

1. Проверьте настройки Restream
2. Убедитесь, что stream key актуален
3. Проверьте статус сервера 89.167.8.169

### Safari/iOS проблемы

1. Откройте ссылку в Safari, а не во встроенном браузере
2. В Telegram: нажмите ⋮ → "Открыть в Safari"
3. Разрешите доступ к камере/микрофону в настройках iOS

### Режиссёрский пульт не подключается

1. Проверьте правильность кода сессии
2. Убедитесь, что студия открыта у ведущего
3. Проверьте интернет-соединение
4. Код меняется при перезагрузке — запросите новый
